# Milano Sensual Congress Flexible Logo

This logo concept is designed as SVG so the annual palette can change without redrawing the mark.

Edit the color variables near the top of each SVG:

```css
--logo-text: #ffffff;
--logo-soft-text: #f8fafc;
--logo-primary: #ec4899;
--logo-secondary: #22c55e;
--logo-gold: #fbbf24;
--logo-shadow: rgba(236, 72, 153, 0.28);
```

Recommended usage:

- `milano-sensual-congress-color-flex-logo.svg`: main navigation, social previews, hero overlays, printed horizontal lockup.
- `milano-sensual-congress-color-flex-mark.svg`: favicon/app icon source, watermark, stickers, profile avatar crop.

Design intent:

- The angled frame keeps continuity with the current congress logo.
- The ribbon mark references Bachata partner flow without locking the logo to one photo or one yearly campaign.
- The wordmark is more readable at small web sizes than the existing raster logo.
- The master logo does not include a year, so the annual refresh can stay limited to palette changes.
